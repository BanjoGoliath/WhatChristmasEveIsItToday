# What Christmas Eve is it Today?

Have you ever wondered what day today was in relevance to Christmas Day? No?
Well I have.

This website tells you what Eve of Christmas the current day is.
For example, the 23rd of December would be Christmas Eve Eve, the 22nd
Christmas Eve Eve Eve, and so forth.

Days between December 25th - 31st are marked as "Christmas Morrow".
